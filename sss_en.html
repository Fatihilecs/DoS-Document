❓ Frequently Asked Questions

When will the game be released?

The exact release date is unknown, but Dominion of States will first launch as a beta. The beta aims to prove the project is real and to raise donations to continue development.

What features will the beta have?

The beta is primarily meant to prove the project exists. There will be a big difference between the beta and full version. The beta will only have limited features, mostly the core mechanics.

What kind of game is Dominion of States?

Dominion of States is a strategy game that includes elements of diplomacy, management, economy, and alternative scenarios. Mod support will also be available so players can create their own scenarios (Modding will not be available in the beta version).

Which platforms will the game be available on?

Beta: Android PC

Full Version: Android iOS Mac PC Linux (not decided)

Which game platforms will it be released on?

Beta: Itch.io

Full Version: PlayStore Steam Itch.io App Store

Will the game be free?

Beta: Free

Full Version: Paid

What are the minimum system requirements?

Minimum requirements are not yet finalized. The goal is for the game to run on both low-end and high-end devices.

Will I be able to create mods?

Yes, the game will support modding. Players will be able to create and share their own scenarios (Modding on mobile is also planned). It is still under discussion whether the beta will support modding.

Will Dominion of States support multiplayer?

Not in the beta. Multiplayer is planned for the full version.

How long will the beta last?

Unless a game-breaking bug is found, there will be no updates during the beta. The focus will shift to developing the full version.

How can I support the game with donations?

Once the system is ready.

Will there be paid DLCs?

Definitely not. DLCs and game features will not be sold for money in the full version.

Will there be content updates?

No for the beta. Yes for the full version. There will be big updates. Spoiler: tanks and navies are planned for the full version.

Will the game have a story?

Definitely not.

Will there be in-game purchases during beta?

No. Only optional donations.

Is Dominion of States similar to Age of History 3 and Europa Universalis IV?

Kind of, yes.
